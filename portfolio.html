Project 1

My first big project that I had while attending FullStack Academy was creating a Zoo html page. This project was to help me practice and show off what I learned about HTML. The requirements for the project were:

  • Include a main header that describes the purpose of the page
  • Include a title for the browser tab
  • Include a heading for each of the animals
  • Add the animal names in a list format
  • Add an external link for each animal
  • Add a photo for each animal
  • Include comments to help others read and understand the code

Project 2

My second big project that I had done while attending FullStack Academy was creating a simple grocery list. This project was to help me practice and show off what I learned about css as well as expand my knowledge of HTML. The requirements for the project were:

  • Incorporate major HTML semantic tags
  • Each category of grocery should be in some type of "section"
  • Sections should have background colors
  • Section headers should have background colors
  • The main header text needs to be a different color
  • Use three types of headings
  • Include two types of lists
  • Use the box model to add borders and spacing to each section
  • Add two photos with padding so it is nicely spaced away from the text element
  • CSS needs to be incorporated using an external CSS file, internal in the header, and inline. I could choose which CSS goes where

Project 3

The third project that I had to do while attending FullStack Academy was to create a pop up message using JavaScript. This was a way to show off what I head learned about JavaScript. The requirements for this project were:

  • Write the pseudocode of your steps and how you will solve this exercise.
  • Create a string for the user saying: You have received this message because you have been chosen to open an important vault. Here is the secret combination:
  • Assign three variables. Each variable contains the corresponding result of calculation using a unique arithmetic operator(+-*/). Each must equal one of the three codes in the combination.
  • Add comments throughout the code to explain each step in the process.
  • Create a dialog box displaying the vault codes and the text if using HTML and a script tag, or create a popup dialog.
